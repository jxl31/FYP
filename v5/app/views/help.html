About Visualisation of DIT's Online Repository

The purpose of Visualisation of DIT's Online Repository is to provide lecturers mainly in Dublin Institute of Technology (DIT) an intuitive view of information such as publications, use of words and coauthorship about researchers in DIT who have published work in various disciplines. The information are presented in visual graphics for a comprehensive overview of the analysed data. The data visualisation used are: Bubble Chart, Network Graph, Bar Chart, Trend Graph , Word Cloud all of which are explained, along with various interactions a user can make with the graph. For a quick guide, go to Quick Tutorial.

Quick Tutorial Back to Top

To start exploring the application go to "Home" and click "Author" or "Discipline" button as shown.

  • Author: user can search through author names
  • Discipline: user can search an author from their disciplines.

A list of author is displayed when "Author" is clicked. A user can also Search by name and set a Filter according to the number of documents an author has and alphabetical order as shown in the image below.

Once an author is selected a user can view their data in different data visualisation. All they have to do is pick whichever they data they want in the available graphs.


Bubble Chart InteractionsBack to Top

- Location

Co-Authorship -> Bubble Chart

- Tooltip

Click on the circle to show a tooltip showing more information about that selected author.


- Go to other author

Double-Click on a bubble to see that author's information.


- Adding Filter

Click the "Add Filter" button and a modal will show. There a user can filter the data by date-coauthored or the number of times the selected author has co-authored with the displayed author. Click "Add" to apply filter or "Cancel" to exit modal.


- Removing Filter

Click the "x" in the applied filter as shown below.


- Hiding/Showing the legend

Click the green "Filter" button on the legend table. When hidden click the "Filter" button again to show the legend.
